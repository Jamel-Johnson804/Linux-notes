## Day 10 Linux Services and Systemctl

## Commands Used
systemctl
sudo systemctl start ssh
sudo systemctl stop ssh
sudo systemctl restart ssh
sudo systemctl status ssh

## What I did
-Checked the status of ssh using sudo systemctl status ssh
-stopped the ssh service using stop ssh
-started the ssh service using start ssh
-restarted the ssh service using restart ssh

## What I learned
systemctl
 -used to manage linux service

status
 -shows whether the service is running or not

stop
 - stops ssh service

start 
 - starts a service

restart 
 -restarts a service
